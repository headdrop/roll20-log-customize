Roll20 로그 커스터마이징 도구

Roll20 로그를 Tistory 신에디터 HTML 편집모드를 사용해 백업하기 위한 커스터마이징 페이지입니다.

이 페이지는 chrome 버전 96.0.4664.45 에서 테스트하였습니다.

반응형 디자인 적용, 주사위 이미지 출력을 원하는 분은 반드시 가이드 페이지를 확인해 주세요.

@HDTRPGver 2.4

HTML 입력

룰 선택

Roll20 기본 표 색상

룰에 따른 주사위 굴림 출력

커스텀 시트 적용

커스텀 시트를 적용하고 싶을 경우, .css 파일을 업로드 해 주세요.

혹은 'CSS 입력' 버튼을 클릭해 css를 직접 입력하세요.

세부 설정

항목에 커서를 올리면 간단한 설명이 나옵니다.

    • 자신이 작성한 메시지의 색이 다른 플레이어가 작성한 메시지의 색과 같아집니다.
    • 메시지 작성 시각이 표시됩니다.
  • info
    • 일반 명령어 굴림(/r 2d6 등)에서 숫자 뒤에 주사위 아이콘을 표시합니다.
    주의:: 티스토리에 적용할 경우, 노션 설명서를 꼭 읽은 후 적용해 주세요.
    • Roll20화면에서 보여지는 폰트를 사용합니다.
    이 옵션을 해제하면, 사용자의 티스토리에 설정된 기본 글꼴로 출력됩니다.
    • 이 옵션을 선택하면 캐릭터별 색상 설정의 글씨 색상이 그 캐릭터의 전체 색상에 적용됩니다. (기본값)
    • 이 옵션을 선택하면 캐릭터별 색상 설정의 글씨 색상이 개별 이름에만 적용됩니다.

    다른 테마 프리셋

    주의 : 다른 테마 프리셋을 적용하거나 기본 테마로 변경하려면 HTML 입력부터 다시 커스텀해야 합니다.

    적용 버튼을 누른 후, 잠시 기다려주세요.

    테마를 선택하면 해당 테마의 미리보기가 이 창에 나타납니다.

    CSS 설정 다운로드

    이 페이지의 css 파일 업로드에 재사용하기 위한 일부 css 코드를 다운로드합니다. 다음 이용시에 이 css 파일을 업로드하면 별도의 설정 없이도 커스텀된 로그를 백업할 수 있습니다.

    업로드한 커스텀 시트, 직접 입력한 CSS 코드, 세부 설정 그리고 캐릭터별 색상 설정이 백업됩니다.
